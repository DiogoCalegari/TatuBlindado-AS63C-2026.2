# Requisitos Funcionais

## RF01 — Cadastro de currículos
O sistema deve permitir o cadastro de currículos contendo informações pessoais, formação acadêmica, experiências profissionais, habilidades e competências.

## RF02 — Edição de currículos
O sistema deve permitir que currículos cadastrados sejam alterados, atualizados ou excluídos.

## RF03 — Consulta de currículos
O sistema deve permitir a visualização dos currículos cadastrados no sistema.

## RF04 — Busca de currículos
O sistema deve permitir buscar currículos por nome, área de atuação, formação, competências ou experiências profissionais.

## RF05 — Filtros de candidatos
O sistema deve permitir filtrar candidatos de acordo com critérios como competências, formação, experiências e área profissional.

## RF06 — Cadastro de funcionários
O sistema deve permitir cadastrar funcionários vinculados à empresa.

## RF07 — Gestão de informações profissionais
O sistema deve permitir armazenar e organizar informações profissionais dos funcionários, como cargo, setor, competências e histórico.

## RF08 — Organização de competências
O sistema deve permitir registrar e identificar as competências profissionais de candidatos e funcionários.

## RF09 — Painel administrativo
O sistema deve disponibilizar um painel administrativo para acompanhar informações gerais sobre currículos, candidatos e funcionários.

## RF10 — Relatórios estratégicos
O sistema deve permitir gerar relatórios para apoiar decisões relacionadas à gestão de pessoas e talentos.
