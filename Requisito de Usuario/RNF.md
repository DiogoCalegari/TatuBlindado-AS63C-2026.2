# Requisitos Não Funcionais

## RNF01 — Usabilidade
O sistema deve possuir uma interface simples, organizada e de fácil utilização para os usuários.

## RNF02 — Responsividade
O sistema deve funcionar corretamente em computadores, tablets e celulares.

## RNF03 — Desempenho
As páginas e consultas do sistema devem carregar em tempo adequado, evitando demora excessiva para o usuário.

## RNF04 — Segurança
As informações de currículos e funcionários devem ser protegidas contra acessos não autorizados.

## RNF05 — Privacidade
O sistema deve tratar dados pessoais de forma responsável, respeitando princípios de privacidade e proteção de dados.

## RNF06 — Compatibilidade
O sistema deve funcionar nos principais navegadores modernos, como Google Chrome, Microsoft Edge e Mozilla Firefox.

## RNF07 — Manutenibilidade
O código do sistema deve ser organizado e documentado para facilitar futuras correções e melhorias.

## RNF08 — Disponibilidade
O sistema deve estar disponível para acesso sempre que necessário, exceto durante manutenções programadas.

## RNF09 — Versionamento
O projeto deve utilizar GitHub para controle de versões e acompanhamento das alterações realizadas.

## RNF10 — Acessibilidade
A interface deve utilizar textos legíveis, bom contraste visual e elementos que facilitem a navegação dos usuários.
