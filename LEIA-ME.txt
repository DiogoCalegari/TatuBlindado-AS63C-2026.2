ARMADILLOHUB - HTML E CSS PURO

Este pacote foi limpo para ter somente front-end estático.

Como abrir:
1. Extraia o ZIP.
2. Abra o arquivo index.html com dois cliques.

Não precisa rodar:
- npm install
- npm run dev
- nodemon
- vite
- node

Arquivos principais:
- index.html
- cadastro.html
- formulario.html
- empresas.html
- hub.html
- perfil-candidato.html
- perfil-gerente.html
- style.css
- assets/

Observação:
Esta versão é visual/estática. Os formulários não salvam dados em banco.
